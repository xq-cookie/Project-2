This project examines the relationship between the **Area Deprivation Index (ADI)** at the county level and the **Gini Index** (state-level income inequality) across the United States. The notebook performs data cleaning, merging, and visualization to explore how socioeconomic deprivation and inequality vary across states.

---

## **Data Sources**

### **1. Area Deprivation Index (ADI) Dataset**

County-level socioeconomic deprivation data.
Source: [https://columbia.redivis.com/datasets/axrk-7jx8wdwc2](https://columbia.redivis.com/datasets/axrk-7jx8wdwc2)

### **2. Gini Index Dataset**

State-level income inequality data (ACS).
Source: [https://columbia.redivis.com/datasets/c204-21pwrwn5h](https://columbia.redivis.com/datasets/c204-21pwrwn5h)

### **3. Merged Dataset**

ADI (county-level) merged with Gini (state-level) using `state_name`, supporting comparative analysis and visualization.
